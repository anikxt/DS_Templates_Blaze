In-Out DP Tree:
indp(node) = Summ. (indp(child) + subtreeSz(child))
outdp(node) = Summ. (outdp(parent) + indp(parent) - (indp(node) - subtreeSz(node)) + (N - subtreeSz(node)))

ans(node) = indp(node) + outdp(node)

   	  root  (outdp(root) = 0) (we build from top to bottom)
